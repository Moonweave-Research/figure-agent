# THE ILLUSTRATOR'S HAND — Engine Architecture (figure-agent piece #3)

The Hand is the agent's set of SVG-manipulation ops. It is SVG-native (SVG-as-truth), drawing on a **two-layer canvas**: an *accuracy-locked structural layer* (the schematic's truth) and a `hand:*` *aesthetic-freedom overlay* drawn strictly on top. Every op widens the existing recipe contract `{id, layer, class, determinism, selector, action{type,params,seed?}, driven_by, rationale, semantic_guard{allowed,reason}}` and routes through `svg_semantic_diff.py` before write. `tikz_patch`/`continue_tikz` are dead enum members. Buildable on `svgpathtools` + `lxml` + `ColorAide` + `kiwisolver` + `fonttools` + `rough-py` (avoid the unmaintained `svgwrite`).

## (1) Operation Library — grouped by craft

**B — Curves & organic form:** `hand.smooth(sel, method=chaikin|catmull_rom, iters)`, `hand.soften_corners(sel, radius, min_angle)` (tangent-arc fillet), `hand.simplify(sel, epsilon, algo=rdp|vw)`, `hand.resample(sel, density)` (shared prep), `hand.roughen(sel, roughness, bowing, seed)` (Rough.js/Wood), `hand.jitter(sel, magnitude, correlation, source=fir|perlin, seed)` (normal-direction, FIR/Perlin-smoothed), `hand.double_stroke(sel, passes, jitter, seed)`, `hand.taper(sel, width_profile)` (centerline→filled outline; SVG has no native variable width), `hand.open_loop(sel, gap_fraction)`.

**A — Depth/shading:** `hand.set_document_light(vector, key, ambient)` (one locked rig), `hand.add_volume_shading(sel, form=sphere|cylinder|plane, light=@doc)` (Lambert 5-stop gradient, fill-only), `hand.add_contact_shadow(sel, softness, opacity)` and `hand.add_long_shadow(sel, angle, length)` and `hand.add_bevel(sel, depth)` (filter-FREE decor geometry), `hand.cel_quantize(sel, bands)`, `hand.add_soft_shadow_filter(...)` (ONLY filter op; auto-sets `color-interpolation-filters=sRGB`; human-gated, rasterizes on PDF export).

**A — Color:** `resolve_token(role)→hex` (frozen SVG token table porting the `.sty` Paul Tol Muted bindings), `derive_shade(hex,ΔL,ΔC,ΔH)` (OKLCH), `set_fill/set_stroke(sel, token|hex)`, `apply_role_gradient(sel, crameri_map)`, `generate_harmony(anchor, scheme)`, plus gates `verify_cvd/verify_grayscale/verify_contrast`.

**Labels/typography:** `measure_text(text, font, pt)` (fonttools, no render), `place_labels(anchors, specs, obstacles, strategy=greedy|annealing, seed)` (Christensen-Marks-Shieber PFLP), `route_leader(anchor, box, obstacles, kind=s|po|opo)` (Bekos boundary-labeling), `set_text_anchor/set_baseline`, `align_labels(axis, edge, tol)`, `render_math(latex)` (MathJax→paths), `format_units(value, unit)`, `enforce_min_print_size(min_pt=5)`, `add_text_halo(sel, color, width)`.

**D — Composition:** `measure(svg)→CompositionReport` (Ngo balance/equilibrium/density + saliency focus + grid candidates, read-only), `snap_to_grid(sel, grid, max_delta)` (kiwisolver, LayoutRectifier), `equalize_gutters`, `set_emphasis(hero, strength)`, `deemphasize`, `reorder_paint`, `draw_overlay(layer=hand:*, ops=[halo|vignette|background_field|group_bracket|common_region|counterweight])`.

## (2) Deterministic Core vs Taste Layer

The split is per-op, declared in `determinism`. **Deterministic** = mechanical, parametrized, reproducible from recipe+seed; executes with *no* critique needed: all geometry (smooth, fillet, resample, simplify, taper, snap_to_grid, route_leader, render_math, format_units, every `verify_*`/`measure_*`, gradient-stop math, OKLCH adjust, PFLP solve). Seeded ops (`roughen`, `jitter`, `double_stroke`, `place_labels` annealing) are deterministic *given the seed*, which is frozen in the decision log — reproducibility = versioned SVG + log, not pixel-lock. **Taste** = needs a critique signal to decide *whether/where/how-much* (`designate_hero`, `set_emphasis` strength, `choose_light_direction`, `assign_role_to_hue`, `choose_harmony_scheme`, `decide_organic_curvature`, `balance_annotation_density`, `tune_corner_softness`, `decide_decoration_restraint`, `shape_whitespace`). **Connection:** a taste op only *proposes* parameters + chosen seed; it then hands off to a deterministic op that executes and logs. So the eye can be wrong about taste without ever producing unreproducible or unverifiable geometry.

## (3) Critique→Edit Loop (reuses figure-agent vocabulary verbatim)

The critique loop (piece #5, ported as-is) runs in the host Claude session and emits the existing signals; each routes to an op — no new enum. **Mechanical routes** (a `MICRO_DEFECT_KIND` deterministically selects an op): `line_crosses_label`/`wire_crosses_label`→`route_leader`; `label_target_detached`/`floating_semantic_cue`→`route_leader` reconnect; `label_path_near_miss`→bounded `place_labels` nudge; `print_scale_unreadable`→`enforce_min_print_size`; a new `edge_misalignment_off_grid`→`snap_to_grid`. **Taste routes** (an `AESTHETIC_ANTIPATTERN_ID`/`LEVER_DIMENSION` fires a taste op): `weak_hero_anchor`+`visual_hierarchy`→`designate_hero`/`set_emphasis`; `dead_flat_vector_finish`+`uniform_line_weight_monotony`+`handcrafted_finish`→`add_volume_shading`/`taper`/`derive_shade`; `childish_shape_language`→*reduce* roughness/revert (the "too sketchy" brake); `cramped_or_dead_whitespace`+`whitespace_breathing`→`shape_whitespace`/`equalize_gutters`; `color_harmony`→palette ops; `annotation_noise_competes_with_science`→`balance_annotation_density`; `cross_panel_grammar`/`panel_style_mismatch`→light/font/grammar coherence; `overdecorated_or_cartoonish`+`maturity_restraint`→`decide_decoration_restraint` (often REJECT). Verdicts `{pass,weak,fail,needs_human}`; routes via `AESTHETIC_GATE_ROUTES`/`SVG_DELTA_ROUTES` `{accept_svg_polish, svg_polish, semantic_backport_required, needs_human_art_direction}`. After write, `rasterize_preview`→re-critique closes the loop via `SVG_DELTA_EVALUATION_STATES` `{improved|no_meaningful_change|regressed|needs_human_art_direction}`; a `regressed→overdecorated`/`label_readability` verdict reverts the edit or lowers the sketchiness scalar.

## (4) Accuracy Lock — the immutable structural guard

`svg_semantic_diff.py` (already pure-SVG) is the seed of piece #6 and gates **every** op before write: invariants are **text identity** (`text_identity_loss`=BLOCKER), **frame/viewBox** (`frame_change`=MAJOR), **element/path/marker inventory**, **color-by-id**, and **transform bounds** (mirrors `MAX_TRANSLATE_PX=10`, stroke ratio `[0.5,2.0]`). The **two-layer split is what makes freedom safe:** structural-layer ops keep those bounds and must leave the inventory byte-invariant — only the 5 sanctioned actions micro-correct truth nodes, and a new `relationship_guard` asserts declared spatial relations (A-left-of-B, A-inside-B, A-connected-by-leader, from native `bbox_inventory_native` replacing the dead `pdftotext` path) survive every translate, rejecting a snap *before* the diff. Aesthetic-freedom ops write only into `<g id="hand:*">`, geometry-bounds-EXEMPT but still passing an **occlusion/detachment guard** (reusing `label_backdrop_overflows_outline`/`label_target_detached`/`floating_semantic_cue` against *locked* bboxes) so an overlay sits visually-above-but-non-interfering and never covers a label or threads between a leader and its target. Decor `defs`/groups are whitelisted by the `hand:*` allowlist so added gradients don't trip `element_inventory_change`/`unsupported_svg_feature`; the sole filter op stays human-gated. `commit()` is **all-or-nothing**: any BLOCKER/MAJOR rejects the whole write (`semantic_backport_required`), never partial. Color edits additionally pass `verify_cvd`/`verify_grayscale`/`verify_contrast` and token-invariance (`semantic_color_remap`) before any write. Thus beautifying is structurally confined — the critique can misjudge taste, but it can never make the schematic scientifically wrong.
