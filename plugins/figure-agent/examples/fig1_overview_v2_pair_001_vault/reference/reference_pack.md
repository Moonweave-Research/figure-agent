## Role Notes

- `topology`: scientific structure that must be preserved, especially Panel A
  linear copolymer topology and Panel C same-matrix trap coexistence.
- `motif`: reusable visual object, such as sulfur ring, DIB ring, trap marker,
  trap level, Gaussian lobe, or hanging cantilever.
- `layout`: panel composition and read path. For this pilot, layout transfer is
  subordinate to the C -> {D, E/F, G} three-spoke evidence structure.
- `style`: palette, line weight, typography, and schematic density. Style
  transfer must not override scientific topology.
- `label_hierarchy`: what text is prominent vs secondary. Panel C and the
  evidence-spoke caption outrank row-2 axis labels.
- `anti_reference`: tempting but scientifically wrong transfer. The Panel A
  network reference is the primary anti-reference for this pilot.
